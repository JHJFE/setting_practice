# setting_practice

Project 설계시, 사용되는 setting과 github repository 사용법 연습용 저장소 입니다.

## Features
- dev 브랜치를 생성해 pr과 merge를 연습합니다.
- issue를 생성하고 milestone으로 그룹 프로세스 진행 현황 및 관련 정보를 관리할 수 있습니다.
- Kanban을 통해 업무를 시각화하여 프로젝트의 효율성을 증가시킵니다.
